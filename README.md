# Bid-Ask Spread LOB Simulator

## Overview
This project explores bid-ask spread dynamics using a stochastic limit order book simulator designed to analyze liquidity, order-flow imbalance, and cancellation intensity in financial markets.

## Features
- Dynamic mid-price evolution
- Stochastic order generation
- Price-time priority matching engine
- Liquidity and imbalance experiments
- Cancellation-rate analysis
- Statistical analysis and visualization

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Repository Structure
- core/ → simulation engine and order book logic
- experiments/ → experiments, analysis, and visualizations
- output/ → generated figures, datasets, and statistics
- paper/ → research paper PDF

## Research Focus
The simulator reproduces stylized market behavior and studies how liquidity and market conditions affect bid-ask spread dynamics.
